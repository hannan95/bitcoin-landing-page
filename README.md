# bitcoin-landing-page
A responsive Bitcoin-themed landing page built using HTML, CSS, JavaScript, and GSAP. The project focuses on modern frontend design, responsive layouts, and interactive animations to create an engaging user experience.

**Features**

•Responsive layout for different screen sizes

•Interactive UI elements and animations

•Smooth transitions powered by GSAP

•Modern landing page structure

•Clean and visually engaging frontend design

**Tech Stack**

•HTML5

•CSS3

•JavaScript

•GSAP

**Purpose**

This project was built to practice responsive frontend development, UI implementation, animations, and interactive user experiences.
